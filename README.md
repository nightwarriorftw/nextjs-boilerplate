## :beginner: About

A next.js boilerplate. Uses : `next.js`, `typscript`, `react`, `tailwindcss`

> Hey stalkers and hackers, Like, share and subscribe :)

## :wrench: Development

### Install Dependencies

```
yarn
```

### Development Server

```
yarn dev
```

### Build App

```
yarn build
```

### Production Server

```
yarn start
```
